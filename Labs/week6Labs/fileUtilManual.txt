Displays the contents of a file.

program filename
program -m filename     Displays the file, 24 lines at a time.

program -?              Displays this help information.

program -l filename     Displays the number of lines in the file.

program -h filename     Displays the first 10 lines of the file.
program -h n filename   Displays the first n lines of the file.

program -t filename     Displays the last 10 lines of the file.
program -t n filename   Displays the last n lines of the file.
